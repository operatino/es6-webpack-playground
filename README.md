# es6-webpack-playground

Support: Latest versions of modern browsers and IE 10+.

## Install instructions

```
git clone https://github.com/operatino/es6-webpack-playground.git && cd es6-webpack-playground
npm i && npm start
open 127.0.0.1:8080
```

## Test

```
npm test
```

## Production build

```
npm run build
```
